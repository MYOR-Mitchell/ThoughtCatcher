# ThoughtCatcher... *"I'll still forget"* 💭 

A backend service that lets you store and manage your thoughts.  
Built with ASP.NET Core and PostgreSQL.

## ✨ Features

- Add new thought 
- Get all thoughts 
- Get thought by ID 
- Update thought 
- Delete thought 
- Sorted for newest thoughts first 

## 🛠️ Future Updates

- A tagging system
- Search & filter
